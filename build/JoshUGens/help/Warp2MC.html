Warp2 MC Warp a multi-channel buffer with a time pointer, with 

window control


Inspired by Chad Kirby's SuperCollider2 Warp1 class, which was inspired by Richard Karpen's sndwarp for CSound. A granular time strecher and pitchshifter. A buffer containing a table to be used for the window must be passed in. 


Warp2MC.ar(numChannels, buffer, pointer, freqScale, windowSize, winBuf, overlaps, windowRandRatio, 

interp, mul, add)


numChannels - number of channels that the buffer will be.

this must be a fixed integer. The architechture of the SynthDef

cannot change after it is compiled.

warning: if you supply a bufnum of a buffer that has a different

numChannels then you have specified to the Warp2MC, it will

fail silently.

buffer - the buffer number of a mono soundfile.

pointer - the position in the buffer.  The value should be between 0 and 1, with 0 being the begining

of the buffer, and 1 the end.

freqScale- the amount of frequency shift. 1.0 is normal, 0.5 is one octave down, 2.0 is one octave up.

Negative values play the soundfile backwards.

windowSIze - the size of each grain window

winBuf - a buffer containing a window table.

overlaps - the number of overlaping windows.

windowRandRatio - the amount of randomness to the windowing function.  Must be between 0 (no

randomness) to 1.0 (probably to random actually)

interp - the interpolation method used for pitchshifting grains. 1 = no interpolation. 2 = linear. 

4 = cubic interpolation (more computationally intensive).



Examples:

s.boot;


(

var winenv;

winenv = Env([0, 1, 1, 0], [0.1, 0.8, 0.1], [5, 0, -5]).asSignal(1024);

winenv.plot; // the window

s.sendMsg(\b_allocRead, 0, "sounds/test.aif", 0, -1);

s.sendMsg(\b_alloc, 1, 1024, 1, [\b_setn, 1, 0, 1024] ++ winenv);


SynthDef(\warp, {arg buffer = 0, winbuf = 1;

var out, pointer, filelength, pitch, env, dir;

pointer = Line.kr(0, 1, 15);

pitch = LFNoise2.kr(1).range(0.25, 4.0).round(0.25).midiratio;

dir = EnvGen.kr(Env([1, 1, -1, -1], [9.999, 0.001, 5]), doneAction: 0);

env = EnvGen.kr(Env([0.01, 1, 1, 0.01], [0.1, 14, 0.9], 'exp'), doneAction: 2);

// control overlaps with MouseX

out = Warp2MC.ar(2, buffer, pointer, pitch, 0.1, winbuf, MouseX.kr(1, 10), 0.05, 2);

Out.ar(0, out);

}).send(s);


)



s.sendMsg(\s_new, \warp, 1000, 0, 0, \buffer, 0, \winbuf, 1);


s.sendMsg(\b_free, 0);

s.sendMsg(\b_free, 1);
